# Sakleaw Studio Website

Учебный проект по дисциплине **Веб-программирование**.

Данный сайт был разработан к экзамену.  
Проект демонстрирует базовые навыки создания современного веб-сайта с использованием HTML, CSS и JavaScript.

---

## Цель проекта

Создать простой сайт IT-студии с основными разделами:

- Главная страница
- Услуги
- Портфолио
- Форма заявки
- Контакты

---

## Используемые технологии

- HTML5
- CSS3
- JavaScript
- GitHub Pages (для хостинга)

---

## Функциональность

Сайт содержит:

- адаптивную верстку
- навигационное меню
- карточки услуг
- галерею проектов
- форму отправки заявки
- кнопку прокрутки вверх
